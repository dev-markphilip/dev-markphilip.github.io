EduCredPH

EduCredPH is a blockchain-powered platform designed to combat credential fraud in higher education. It was developed as part of my doctoral dissertation.

Project Description

Designed and developed a tamper-proof credential verification platform using Hyperledger Fabric to combat document fraud. The system enables organizations to issue and verify credentials securely by leveraging SHA-256 hashing and blockchain immutability, allowing real-time public verification via unique hashes or QR codes.

The platform provides a fraud-resistant solution for institutions issuing diplomas, certificates, or licenses, reducing manual verification efforts while enhancing trust through transparent, real-time validation.

Key Features & Responsibilities

  • Blockchain-Backed Security: Implemented a decentralized ledger to ensure credential integrity
  • Hash-Based Verification: Generated and stored SHA-256 hashes of sensitive documents
  • Public Lookup Portal: Enabled verification via web interface or QR code scanning
  • RESTful API Development: Built scalable backend services using Node.js & Express.js
  • Thorough API Testing: Validated endpoints using Postman for reliability

Project information

  • Project Duration 2024-present
  • Technologies: Node.js, Express.js, Hyperledger Fabric, SHA-256, Linux
